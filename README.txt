The following is the documentation on how to use the shot counter device and the web application.  

RTC Shield Battery: 1216 

Getting Data Wirelessly from Shot Counter Device: 

Plug Xbee into USB port on computer, it is important that this is plugged into the same USB port every time.  

Click on the webapp.cmd file this will be located on your Desktop. This should automatically launch the web application.  

Get within 20 yards of Shot Counter Device. 

Click on the Get Data tab in the web application. Wait a minute or two you should see the data that the device had stored displayed on this web page.
